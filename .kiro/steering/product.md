# Product Overview

## Nuraziz - Athletic Coach & Time Management Application

A comprehensive fitness and lifestyle management platform with AI-powered features, designed for Uzbek-speaking users. The application combines workout tracking, nutrition guidance, goal setting, and administrative management in a single platform.

## Core Features

- User authentication with admin approval workflow
- Personalized workout planning and tracking
- AI-powered goal setting and workout plan generation
- Nutrition analysis and meal planning
- Daily schedule management (school, work, future planning)
- Video consultation and AI chat support
- Admin panel for user management and analytics
- MongoDB-based data persistence with file storage fallback

## User Types

1. **Regular Users**: Access fitness tracking, AI features, scheduling, and personal management tools
2. **Admin Users**: Full system access with user approval, subscription management, and analytics dashboard

## Language

Primary language is Uzbek (O'zbek tili) with UI text, notifications, and AI responses in Uzbek.
